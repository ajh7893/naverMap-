오시는 길

경기도 광주시 남종면 산수로 1692 경기도 팔당전망대
TEL. 031-8008-6933

대중교통 이용시

01서울에서 오시는 길

  • 서울 → 퇴촌 동서울터미널기점(13-2번) – 천호동 – 신장(하남) – 번천 – 퇴촌
  • 첫차 05:40 / 막차 22:40 (배차간격 20~25분)
  • 퇴촌 → 남종면 분원리 자율순환버스 이용 : 8분 소요
  • 분원리 – 07:19 / 08:22 / 10:29 / 11:22 / 15:09 / 16:22 / 17:09 / 18:22 / 21:12 / 21:51

02광주에서 오시는 길

  • 광주 → 퇴촌 광주 축협앞에서 퇴촌(남종)행 버스이용
  • 배차시간 – 06:20 / 08:15 / 11:30 / 12:40 / 19:10
  • 퇴촌 → 남종면 분원리 구간 동일

03팔당역에서 오시는 길

  • 팔당역에서 6번 국도 서울방향으로 200m 직진후 팔당대교를 건너서 45번 국도 광주방향으로 좌회전하여 10Km 직진
  • 퇴촌방향(도마삼거리)으로 좌회전하여 2Km 직진 → 좌회전하여 5Km 직진 → 남종면 분원리 → 경기도수자원본부(팔당전망대)
자가용 이용시

네비게이션 ‘경기도수자원본부’ 검색

  • 경안IC 나와서 우회전 → 도마삼거리 우회전 → 퇴촌사거리 좌회전하여 직진 → 분원리 둔치 → 광동하수 처리장 앞에서 342번 지방도로 남종/분원방향 좌회전 → 경기도수자원본부
  • 미사리 방면에서 오시는 경우, 45번 국도를 직진하여 (배알미대교 통과) 도마삼거리에서 우회전하여 직진
